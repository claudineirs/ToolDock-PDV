# 🧾 ToolDock PDV

Sistema de ponto de venda (PDV) offline desenvolvido em **HTML, CSS e JavaScript puro**, com foco em simplicidade, usabilidade e funcionamento local sem necessidade de servidor.

---

## 🚀 Sobre o Projeto

O **ToolDock PDV** é uma solução leve e funcional para pequenos negócios que precisam de um sistema de vendas simples, rápido e que funcione totalmente offline.

Ideal para:

* Pequenos comércios
* Lojas locais
* Uso pessoal ou demonstração
* Base para projetos maiores

---

## ⚙️ Funcionalidades

### 🛒 Vendas

* Lançamento rápido de produtos
* Busca por nome ou código
* Controle de quantidade e preço
* Cálculo automático de total, troco e pagamento

### 📦 Produtos

* Cadastro completo de produtos
* Código gerado automaticamente
* Controle de estoque
* Alerta de estoque mínimo
* Edição e exclusão

### 👤 Clientes

* Cadastro de clientes
* Seleção na venda
* Edição e exclusão

### 💰 Caixa

* Abertura de caixa com valor inicial
* Controle de vendas por sessão
* Separação por forma de pagamento:

  * Dinheiro
  * PIX
  * Cartão
* Registro de movimentos:

  * Suprimento
  * Sangria
* Impressão da movimentação do caixa
* Controle por usuário logado

### 📊 Histórico

* Listagem de vendas
* Filtros por pagamento e status
* Cancelamento de vendas
* Estatísticas do dia

### 💾 Backup e Restore

* Exportação de dados em JSON
* Importação de backup
* Segurança de dados local

### 🖨️ Impressão

* Impressão em formato:

  * A4
  * Bobina 80mm

## ▶️ Como Executar

1. Baixe ou clone o repositório:

```bash
git clone https://github.com/Claudineirs/ToolDock-PDV.git
```

2. Abra a pasta do projeto

3. Execute o sistema:

* Clique duas vezes no arquivo `index.html`
  ou
* Abra com seu navegador

## ⚠️ Requisitos

* Navegador moderno (Chrome, Edge, Firefox)
* Não requer internet após carregamento
* Não requer instalação

---

## 🧠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage (armazenamento local)

---

## 📌 Regras importantes do sistema

* O caixa precisa estar aberto para realizar vendas
* O caixa pertence ao usuário que abriu
* Os dados são armazenados localmente no navegador
* Backup deve ser feito manualmente pelo usuário
